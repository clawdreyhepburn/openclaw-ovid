# @clawdreyhepburn/openclaw-ovid

OpenClaw plugin for [OVID](https://github.com/clawdreyhepburn/ovid) agent identity.

## Tools

- **`ovid_mint`** — Mint an OVID token for a sub-agent
- **`ovid_verify`** — Verify an OVID token
- **`ovid_inspect`** — Decode and pretty-print an OVID token (without verification)

## CLI

```bash
openclaw ovid status   # Show keypair status
openclaw ovid keygen   # Generate/regenerate orchestrator keypair
```

## Install

```bash
openclaw plugins install @clawdreyhepburn/openclaw-ovid
```
